---
title: modal.Volume
description: API reference for Modal Volume class
---

A writeable volume that can be used to share files between one or more Modal functions.

The contents of a volume is exposed as a filesystem. You can use it to share data between different functions, or to persist durable state across several instances of the same function.

Unlike a networked filesystem, you need to explicitly reload the volume to see changes made since it was mounted. Similarly, you need to explicitly commit any changes you make to the volume for the changes to become visible outside the current container.

## Usage

```python
import modal

app = modal.App()
volume = modal.Volume.from_name("my-persisted-volume", create_if_missing=True)

@app.function(volumes={"/root/foo": volume})
def f():
    with open("/root/foo/bar.txt", "w") as f:
        f.write("hello")
    volume.commit()  # Persist changes

@app.function(volumes={"/root/foo": volume})
def g():
    volume.reload()  # Fetch latest changes
    with open("/root/foo/bar.txt", "r") as f:
        print(f.read())
```

    name: str,

Reference a Volume by name, creating if necessary.

## Methods

### volume.commit

```python
volume.commit()
```

Persist changes made to the volume.

This must be called after writing files to make them visible to other containers.

**Example:**

```python
@app.function(volumes={"/data": volume})
def write_data():
    with open("/data/output.txt", "w") as f:
        f.write("results")
    volume.commit()
```

### volume.reload

```python
volume.reload()
```

Fetch the latest changes to the volume.

This must be called to see changes made by other containers.

**Example:**

```python
@app.function(volumes={"/data": volume})
def read_data():
    volume.reload()
    with open("/data/output.txt", "r") as f:
        print(f.read())
```

### volume.listdir

```python
volume.listdir(
    path: str,
) -> list[FileEntry]
```

List files in a directory on the volume.

<ParamField path="path" type="str" required>
  Path to list (can include glob patterns).
</ParamField>

<ResponseField name="entries" type="list[FileEntry]">
  List of file entries.
</ResponseField>

**Example:**

```python
entries = volume.listdir("/data")
for entry in entries:
    print(entry.path, entry.type, entry.size)
```

### volume.read_only

```python
volume.read_only() -> Volume
```

Configure Volume to mount as read-only.

<ResponseField name="volume" type="Volume">
  New read-only Volume instance.
</ResponseField>

**Example:**

```python
volume = modal.Volume.from_name("my-volume", create_if_missing=True)

@app.function(volumes={"/mnt/items": volume.read_only()})
def f():
    with open("/mnt/items/my-file.txt") as f:
        return f.read()
```
